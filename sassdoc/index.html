    font-sizes

    $font-sizes: (
    "xs": clamp(0.8rem, 1.5vw, 1rem),
    "s": clamp(0.9rem, 2vw, 1.125rem),
    "m": clamp(1rem, 2.5vw, 1.25rem),
    "l": clamp(1.25rem, 3vw, 1.75rem),
    "xl": clamp(1.5rem, 4vw, 2.5rem),
    "xxl": clamp(2.5rem, 6vw, 5rem)
);

    Description

    Escala tipográfica fluida con clamp().

    Type

    Map

    Map structure

    Map key NameMap key DescriptionMap key TypeMap key Value
    xs

    Texto muy pequeño. Ej: labels, pie de página.

    		Lengthclamp(0.8rem, 1.5vw, 1rem)
    s

    Texto pequeño. Ej: citas, descripciones cortas.

    		Lengthclamp(0.9rem, 2vw, 1.125rem)
    m

    Cuerpo de texto general.

    		Lengthclamp(1rem, 2.5vw, 1.25rem)
    l

    Subtítulos, taglines.

    		Lengthclamp(1.25rem, 3vw, 1.75rem)
    xl

    Títulos de sección (h2, h3).

    		Lengthclamp(1.5rem, 4vw, 2.5rem)
    xxl

    Título hero principal (h1).

    		Lengthclamp(2.5rem, 6vw, 5rem)

    input-validation

    @mixin input-validation() { ... }

    Description

    Aplica estilos de validación al input según su estado. Usa las pseudoclases :valid e :invalid para detectar el contenido sin necesidad de JavaScript. El :not(:placeholder-shown) evita que los estilos se apliquen en inputs vacíos al cargar la página.

    Parameters

    None.

    Example

    .reserva__input {
  @include input-validation;
}

    Output

    Borde verde cuando el input es válido, rojo cuando es inválido.

    Requires

    • [function] color
    • [function] color

    Author

    • rverdejomartin
